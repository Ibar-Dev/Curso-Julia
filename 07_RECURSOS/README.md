# Recursos y ejemplos útiles para Julia

Esta carpeta contiene ejemplos temáticos, glosario, errores frecuentes, patrones comunes y recursos adicionales para tu aprendizaje.

## Índice de recursos

- ejemplo_broadcasting.jl — Uso del broadcasting
- ejemplo_despacho_multiple.jl — Despacho múltiple
- ejemplo_funcion_anonima.jl — Funciones anónimas
- ejemplo_macros.jl — Macros en Julia
- ejemplo_manejo_errores.jl — Manejo de errores
- ejemplo_map_do.jl — Uso de map y do
- ejemplo_mutabilidad.jl — Mutabilidad
- ejemplo_operador_punto.jl — Operador punto
- ejemplo_pares_clave_valor.jl — Pares clave-valor
- ejemplo_scope.jl — Ámbito de variables
- ejemplo_sintaxis_extra.md — Sintaxis especial
- errores_frecuentes.md — Errores comunes y soluciones
- glosario.md — Glosario de términos
- patrones_comunes.md — Patrones de buenas prácticas
- plantilla_proyecto.md — Plantilla de organización de proyectos
- recursos_adicionales.md — Enlaces y libros recomendados

## Ruta de aprendizaje sugerida

1. Consulta el glosario y patrones comunes antes de comenzar cada sección.
2. Revisa los ejemplos temáticos para entender conceptos clave.
3. Si encuentras un error, busca en errores_frecuentes.md.
4. Usa la plantilla_proyecto.md para organizar tus propios proyectos.
5. Explora recursos_adicionales.md para profundizar.

## Recomendaciones

- Vuelve a esta carpeta siempre que tengas dudas o quieras repasar conceptos.
- Comenta y documenta tus propios ejemplos inspirándote en los de aquí.
- Comparte tus aprendizajes y dudas con la comunidad.
