# Conceptos Intermedios - Nivel Básico

Bienvenido a la sección de conceptos intermedios (básico) de Julia. Aquí encontrarás ejercicios diseñados para profundizar en temas como comprensión de listas, manejo de archivos y funciones anidadas.

## Índice de ejercicios

- 001_comprension_listas.jl — Comprensión de listas

## Ruta de aprendizaje sugerida

1. Repasa los fundamentos de Julia (variables, tipos, estructuras de control).
2. Realiza los ejercicios de comprensión de listas y experimenta con tus propios ejemplos.
3. Lee y escribe archivos simples para practicar entrada/salida.
4. Explora funciones anidadas y su utilidad.

## Recomendaciones

- Lee cuidadosamente la teoría y los ejemplos antes de intentar el TODO.
- No copies el ejemplo, ¡intenta resolver el ejercicio por tu cuenta!
- Consulta la documentación oficial de Julia si tienes dudas.
- Comenta tu código y explica tus decisiones.
