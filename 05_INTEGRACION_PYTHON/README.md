# Integración Julia-Python: Ejercicios prácticos

Esta sección te permite practicar la interoperabilidad entre Julia y Python usando PyCall y PythonCall.

## Índice de ejercicios

- pycall_basico.jl — Ejemplo y TODO con PyCall
- pythoncall_basico.jl — Ejemplo y TODO con PythonCall

## Ruta de aprendizaje sugerida

1. Repasa los ejemplos de integración con PyCall y PythonCall.
2. Realiza los TODO propuestos en cada archivo.
3. Consulta el glosario y patrones comunes si encuentras dificultades.
4. Experimenta integrando otros módulos de Python.

## Recomendaciones

- Lee la teoría y los ejemplos antes de resolver el TODO.
- No copies el ejemplo, ¡intenta resolver el ejercicio por tu cuenta!
- Comenta tu código y explica tus decisiones.
- Consulta la documentación oficial de Julia y Python si tienes dudas.
- Revisa el glosario y patrones comunes en 07_RECURSOS.
