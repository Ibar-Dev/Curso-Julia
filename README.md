# Julia Curso Completo

Este curso unifica y mejora los contenidos de "Introduccion_Julia" y "Tutorial_Julia". La estructura está organizada por niveles y recursos, eliminando redundancias y facilitando el aprendizaje progresivo.

## Estructura
- **00_PREREQUISITOS**: Instalación, introducción y comparación con Python.
- **01_FUNDAMENTOS**: Variables, tipos, operaciones básicas, estructuras de datos, condicionales, bucles, funciones, structs, manejo de errores, módulos y patrones idiomáticos.
- **02_CONCEPTOS_INTERMEDIOS**: Ejercicios y proyectos intermedios.
- **03_TRANSICION_AVANZADA**: Ejercicios avanzados, performance, type stability, macros, manejo de archivos y paquetes externos.
- **04_CONCEPTOS_AVANZADOS**: Temas avanzados y ejemplos prácticos.
- **07_RECURSOS**: Glosario, errores frecuentes, patrones comunes y recursos adicionales.

## Cómo usar
1. Avanza por los directorios según tu nivel.
2. Consulta los ejercicios y ejemplos para practicar.
3. Revisa los recursos para resolver dudas y mejorar tu comprensión.
