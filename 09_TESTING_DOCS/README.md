## Ruta de aprendizaje sugerida

1. Repasa los ejemplos de testing y documentación.
2. Realiza los TODO propuestos en cada archivo.
3. Consulta el glosario y patrones comunes si encuentras dificultades.

## Recomendaciones

- Lee la teoría y los ejemplos antes de resolver el TODO.
- No copies el ejemplo, ¡intenta resolver el ejercicio por tu cuenta!
- Comenta tu código y explica tus decisiones.
- Consulta la documentación oficial de Julia si tienes dudas.
- Revisa el glosario y patrones comunes en 07_RECURSOS.
