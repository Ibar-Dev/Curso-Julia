# Configuración y uso de módulos propios en Julia

Esta carpeta contiene ejemplos de cómo crear, importar y documentar módulos y submódulos en Julia.

## Índice de ejemplos

- MiModulo.jl — Ejemplo de módulo propio
- ejemplo_uso_modulo.jl — Cómo importar y usar un módulo
- Project.toml — Dependencias del módulo

## Ruta de aprendizaje sugerida

1. Lee el ejemplo de MiModulo.jl y entiende su estructura.
2. Prueba importar el módulo desde ejemplo_uso_modulo.jl.
3. Modifica el módulo agregando nuevas funciones y documenta los cambios.
4. Consulta el glosario y patrones comunes si tienes dudas.

## Recomendaciones

- Comenta y documenta cada función del módulo.
- Usa Project.toml para gestionar dependencias.
- Escribe pruebas para tus módulos en una carpeta test/.
- Consulta la documentación oficial de Julia sobre módulos.
