# Mini-proyectos en Julia: Retos y colaboración

Esta sección contiene proyectos aplicados y retos de integración para poner en práctica todo lo aprendido.

## Índice de proyectos

- analisis_datos_pluto.jl — Mini-proyecto de análisis de datos
- reto_integracion.jl — Reto de integración de conceptos
- proyecto_colaborativo.md — Ideas para proyectos grupales

## Ruta de aprendizaje sugerida

1. Elige un mini-proyecto y resuélvelo paso a paso.
2. Realiza el reto de integración para combinar varios temas.
3. Propón o participa en un proyecto colaborativo.
4. Consulta el glosario y patrones comunes si encuentras dificultades.

## Recomendaciones

- Lee la teoría y los ejemplos antes de resolver el TODO.
- No copies el ejemplo, ¡intenta resolver el proyecto por tu cuenta!
- Comenta tu código y explica tus decisiones.
- Consulta la documentación oficial de Julia si tienes dudas.
- Revisa el glosario y patrones comunes en 07_RECURSOS.
