# Visualización en Julia: Ejercicios prácticos

Esta sección te permite practicar la creación de gráficos básicos y avanzados usando Plots.jl y Makie.jl.

## Índice de ejercicios

- plots_basico.jl — Ejemplo y TODO con Plots.jl
- makie_avanzado.jl — Ejemplo y TODO con Makie.jl

## Ruta de aprendizaje sugerida

1. Repasa los ejemplos de visualización con Plots y Makie.
2. Realiza los TODO propuestos en cada archivo.
3. Consulta el glosario y patrones comunes si encuentras dificultades.
4. Experimenta creando tus propios gráficos.

## Recomendaciones

- Lee la teoría y los ejemplos antes de resolver el TODO.
- No copies el ejemplo, ¡intenta resolver el ejercicio por tu cuenta!
- Comenta tu código y explica tus decisiones.
- Consulta la documentación oficial de los paquetes si tienes dudas.
- Revisa el glosario y patrones comunes en 07_RECURSOS.

---

¡Practica la visualización y comunica tus datos de forma efectiva!
