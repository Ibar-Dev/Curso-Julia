# ETL de Datos: Ejercicios prácticos

Esta sección te permite practicar la extracción, transformación y carga de datos (ETL) usando Julia con diferentes formatos.

## Índice de ejercicios

- etl_csv.jl — Ejemplo y TODO con CSV
- etl_excel.jl — Ejemplo y TODO con Excel
- etl_json.jl — Ejemplo y TODO con JSON

## Ruta de aprendizaje sugerida

1. Repasa los ejemplos de ETL para cada formato.
2. Realiza los TODO propuestos en cada archivo.
3. Consulta el glosario y patrones comunes si encuentras dificultades.
4. Experimenta con tus propios archivos de datos.

## Recomendaciones

- Lee la teoría y los ejemplos antes de resolver el TODO.
- No copies el ejemplo, ¡intenta resolver el ejercicio por tu cuenta!
- Comenta tu código y explica tus decisiones.
- Consulta la documentación oficial de Julia y los paquetes si tienes dudas.
- Revisa el glosario y patrones comunes en 07_RECURSOS.
